How to run the Expense Management System in PHP and MySQL 

1. Download the zip file

2. Extract the file and copy carwash folder

3. Paste inside root directory
For laravel = (laragon laragon/www)
For xampp = (xampp xampp/htdocs)
For wampp = (wamp wamp/www)
For lamp = (for lamp var/www/html)


4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name expensedata

6. Import expensedata.sql file(Given inside the zip package in SQL file folder)

7. Open your favorite browser and search this link

For User = Run the script http://localhost/expense
